Build Instructions
------------------

- Check out products/surveys.designer and add it as an Eclipse project
- Check out purcforms/surveys.designer.sharedlib to a folder called surveys.designer.sharedlib that sits in the same directory as the surveys.designer folder
- Add it as an eclipse project called SharedLib
- From eclipse Package Explorer, right click on surveys.designer/build.xml and go to "Run As">"2 Ant Build"
- From the pop up window, check the option "gwtc" and uncheck the option "dev[default]" then finish by clicking "Apply" 
- If the build completes without any errors, the compilied application can be obtained from the war folder of this project
- Copy these files from the war directory to the appropriate directory on your webserver.
